## 📝 **TROUBLESHOOTING**

### If you see errors:

1. **Clean and rebuild:**
   ```bash
   flutter clean
   flutter pub get
   flutter run
   ```

2. **Check device connection:**
   ```bash
   adb devices
   ```

3. **Restart ADB:**
   ```bash
   adb kill-server
   adb start-server
   ```

4. **Check Flutter doctor:**
   ```bash
   flutter doctor
   ```
